## 🇩🇪 **PMC Virtual Eyes — Germany** *(Frontend, Flutter Web)*

### 🧭 Overview  
Flutter Web frontend for **PMC — Precise Meta Controlling** (Virtual Eye GmbH), delivered as a Germany-only outsourcing project. Single-page application focused on budget steering with plan/actual comparisons, forecast and reporting views, a centralized control layer, and audit-safe workflows.

### ✨ Features
- Spreadsheet-like tables for plan/actual comparisons  
- Forecast and reporting dashboards  
- Real-time data refresh with status indicators  
- Integration with SAP, MS Project, Oracle, Ivalua  
- Audit trail and change history visualizations  
- Role-based navigation and permissions  
- Responsive layout with accessibility compliance  

### 👥 Audience  
Finance, Controlling, and PMO teams in large enterprises.

### 🌍 Localization  
**de-DE**

### 💻 Platform  
Flutter Web (SPA)

### 🧰 Tech Stack
